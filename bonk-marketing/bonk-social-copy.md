## LinkedIn Post (Long)

Every professional has a prep ritual.

For DJs, that ritual has always been unnecessarily painful.

Bonk is the local-first DJ library workstation — the tool you open *before* a gig to clean, analyze, and plan, not during performance.

**What it does:**

- Direct Rekordbox 6 and 7 master.db import/export — no XML intermediary, works with encrypted databases
- AutoTag from MusicBrainz, Spotify, Discogs, Beatport, and iTunes
- VibeNet: ML-powered energy/danceability/mood scoring (0–10) for every track
- Graph Playlist: harmonic set planning with Camelot wheel compatibility
- Chapter Builder: structured set timeline with chapter markers
- Preflight Center: missing file relink, duplicate merge, and smart fixes
- Batch Genre Management Suite
- Format conversion: FLAC/AIFF/MP3/WAV/M4A
- Watch Folder automation
- Command Palette (Cmd+K)
- Library snapshots with atomic write backup
- Cross-platform: macOS, Windows, Linux

**The positioning is intentional.** Bonk is not a performance tool. It's the tool that makes sure your performance doesn't start with a 20-minute metadata nightmare.

Built for DJs who live in Rekordbox, Serato, or Engine DJ — and anyone with a large music collection that deserves proper metadata hygiene.

Available now at bonkmusic.qzz.io

# Bonk! Launches — The Local-First DJ Library Workstation

## Metadata is where the real work happens

Every DJ knows the feeling. You show up to a gig, load up your set, and spend the first 20 minutes fighting with bad metadata — tracks with the wrong key, missing album art, duplicate entries, broken file paths. The performance suffers because the prep work wasn't done.

The problem isn't the music. It's the metadata.

Bonk exists because DJs deserve tools that treat their library with the same seriousness they treat their mixes.

---

## The Problem

DJs spend hours before every gig on tasks that should take minutes:

- Cleaning up metadata across Rekordbox, Serato, and Engine DJ databases
- Exporting/importing through fragile XML intermediaries
- Manually tagging tracks because automatic enrichment doesn't exist for DJ music
- Planning set flow with no visual harmonic or energy guidance
- Hunting down missing files and duplicates before a show

The tools DJs use during a performance are polished. The tools they use *before* a performance are not.

---

## The Solution

**Bonk!** is a local-first DJ library workstation for cleaning, analyzing, and planning sets. It runs entirely on your machine — no cloud dependency, no subscription, no data leaving your network. Import your existing Rekordbox 6 or 7 master.db directly, including encrypted databases. Make your changes. Export back. Done.

---

## Key Features

**Import & Export**
Direct Rekordbox 6/7 master.db import and export — no XML intermediary, works with encrypted databases. Supports FLAC, AIFF, MP3, WAV, and M4A.

**Quick Tag**
Isolated single-folder tagging workflow. Tag one folder fast without touching the rest of your library.

**AutoTag**
Enrich metadata from MusicBrainz, Spotify, Discogs, Beatport, and iTunes simultaneously. One track, multiple sources, unified result.

**VibeNet Audio Features**
ML-powered energy, danceability, and mood scoring on a 0–10 scale. See the emotional shape of your set before you play it.

**Graph Playlist**
Visual harmonic set planning with Camelot wheel compatibility. Build sets that transition smoothly without guessing.

**Chapter Builder**
Structured set timeline with chapter markers. Break your set into movements and navigate it like a document.

**Analytics**
Genre, key, BPM, and artist distribution charts. Know your library's composition at a glance.

**Preflight Center**
Missing file relink, duplicate merge, and smart fixes — all in one checklist before you head to the venue.

**Batch Genre Management Suite**
Bulk genre operations across your entire library with intelligent conflict resolution.

**Format Conversion**
Convert between FLAC, AIFF, MP3, WAV, and M4A without leaving the app.

**Watch Folder**
Automated metadata processing when you drop new tracks into a watched folder.

**Command Palette**
Cmd+K to access everything. No hunting through menus.

**Library Snapshot + Atomic Write Backup**
Point-in-time snapshots of your entire library. Atomic writes ensure your database is never corrupted by a partial save.

**Cross-Platform**
macOS, Windows, and Linux. One workflow across every OS.

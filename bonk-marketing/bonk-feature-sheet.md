# Bonk! Feature Sheet

## Import

| Feature | Description | Format Support |
|---|---|---|
| Rekordbox Direct Import | Read Rekordbox 6/7 master.db directly, including encrypted databases | .db |
| Watch Folder | Automated import and metadata processing when new files are dropped | FLAC, AIFF, MP3, WAV, M4A |
| Batch File Import | Multi-select import with intelligent duplicate detection | FLAC, AIFF, MP3, WAV, M4A |

## Edit

| Feature | Description | Format Support |
|---|---|---|
| Quick Tag | Isolated single-folder tagging workflow — fast, targeted edits | All |
| Batch Genre Management Suite | Bulk genre operations with conflict resolution | All |
| Metadata Editor | Full ID3v2/Vorbis/M4A tag editing with undo support | FLAC, AIFF, MP3, WAV, M4A |
| Duplicate Merge | Identify and merge duplicate tracks across the library | All |
| Missing File Relink | Detect broken references and re-link to new file locations | All |

## Analyze

| Feature | Description | Format Support |
|---|---|---|
| VibeNet Audio Features | ML-powered energy/danceability/mood scoring (0–10 scale) | FLAC, AIFF, WAV |
| BPM Detection | High-precision beat grid and BPM analysis | FLAC, AIFF, WAV |
| Key Detection | Algorithmic key detection with Camelot notation | FLAC, AIFF, WAV |
| AutoTag | Multi-source enrichment: MusicBrainz, Spotify, Discogs, Beatport, iTunes | All |
| Analytics Dashboard | Genre/key/BPM/artist distribution charts | All |

## Plan

| Feature | Description | Format Support |
|---|---|---|
| Graph Playlist | Visual harmonic set planning with Camelot wheel compatibility | All |
| Chapter Builder | Structured set timeline with chapter markers | All |
| Set Timeline View | Time-ordered track list with energy and key overlays | All |
| Energy Flow Visualization | See the energy curve of your set before playing | All |

## Export

| Feature | Description | Format Support |
|---|---|---|
| Rekordbox Direct Export | Write back to Rekordbox 6/7 master.db directly | .db |
| Format Conversion | Convert between FLAC/AIFF/MP3/WAV/M4A | FLAC, AIFF, MP3, WAV, M4A |
| Playlist Export | Export playlists as M3U, PLS, or JSON | All |
| Library Snapshot | Point-in-time snapshot of entire library state | All |
| Atomic Write Backup | Safe write mechanism — no partial saves or corruption | All |

# Bonk! — One-Page Product Summary

## What It Is

Bonk is a local-first DJ library workstation for cleaning, analyzing, and planning sets — the tool you open *before* a gig, not during it.

---

## Who It's For

- **Rekordbox power users** who need direct .db access without XML intermediaries
- **Music producers** managing large sample and WIP libraries that need the same metadata discipline as a DJ set
- **Anyone with a large music collection** that has grown organically and needs systematic metadata hygiene

---

## Top 6 Features

1. **Rekordbox Direct Import/Export** — read and write master.db files directly, including encrypted DBs from Rekordbox 6 and 7
2. **Quick Tag** — isolated single-folder workflow for fast, targeted metadata edits without loading your full library
3. **AutoTag** — enrich from MusicBrainz, Spotify, Discogs, Beatport, and iTunes in one pass
4. **Chapter Builder** — structured set timeline with chapter markers and Camelot key filtering
5. **Graph Playlist** — visual harmonic set planning with Camelot wheel compatibility and energy flow
6. **VibeNet Audio Features** — ML-scored energy/danceability/mood 0–10 for every track

---

## How It Works

**1. Import**
Drop in your Rekordbox master.db or connect a watched folder or import a custom. Bonk reads your existing library structure — no re-tagging from scratch.

**2. Clean, Analyze, Plan**
Use Quick Tag for isolated folders, Chapter Builder for structured set timelines, and Graph Playlist for harmonic ordering. Run AutoTag and VibeNet for enrichment, then Preflight to catch issues before export.

**3. Export / Sync**
Write back to Rekordbox directly, export to any supported format, or let Watch Folder handle new imports automatically.

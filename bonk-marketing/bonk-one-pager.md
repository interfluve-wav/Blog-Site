# Bonk! — One-Page Product Summary

## What It Is

Bonk is a local-first DJ library workstation for cleaning, analyzing, and planning sets — the tool you open *before* a gig, not during it.

---

## Who It's For

- **Rekordbox/Serato/Engine DJ power users** who need direct .db access without XML intermediaries
- **Music producers** managing large sample and WIP libraries that need the same metadata discipline as a DJ set
- **Anyone with a large music collection** that has grown organically and needs systematic metadata hygiene

---

## Top 5 Features

1. **Rekordbox Direct Import/Export** — read and write master.db files directly, including encrypted DBs from Rekordbox 6 and 7
2. **AutoTag** — enrich from MusicBrainz, Spotify, Discogs, Beatport, and iTunes in one pass
3. **VibeNet Audio Features** — ML-scored energy/danceability/mood 0–10 for every track
4. **Graph Playlist** — harmonic set planning with Camelot wheel visualization
5. **Preflight Center** — missing file relink, duplicate merge, and smart fixes before every gig

---

## How It Works

**1. Import**
Drop in your Rekordbox master.db or connect a watched folder. Bonk reads your existing library structure — no re-tagging from scratch.

**2. Clean, Analyze, Plan**
Use Quick Tag for isolated folders, AutoTag for enrichment, VibeNet for audio features, and Graph Playlist for harmonic set planning. Run Preflight to catch issues.

**3. Export / Sync**
Write back to Rekordbox directly, export to any supported format, or let Watch Folder handle new imports automatically.

## Three Key Value Props

**1. Direct Rekordbox integration — no intermediary**
Bonk reads and writes your Rekordbox 6/7 master.db directly, including encrypted databases. No XML export/import. No broken playlists. No lost cue points.

**2. Your library, fully analyzed**
AutoTag enriches from MusicBrainz, Spotify, Discogs, Beatport, and iTunes simultaneously. VibeNet scores every track's energy, danceability, and mood on a 0–10 scale using an ML model built for DJ music. See your library's composition before you play it.

**3. Preflight before every gig**
Missing file relink, duplicate merge, smart fixes — all in one checklist. Bonk catches what would have ruined your set *after* you were already supposed to be playing.

## Feature Callout

**Graph Playlist + Chapter Builder**

Build your set visually.

Graph Playlist shows harmonic transitions on a Camelot wheel — pick tracks that flow, not just by BPM but by key compatibility.

Chapter Builder lets you structure your set into movements. Tag energy curves, name chapters, navigate your set like a document.

Play the set you planned, not whatever happened to load.
